## 1. 产品概述

「暗影地牢」是一款基于浏览器的 Roguelike 地牢探索游戏，玩家在程序化随机生成的地牢中探索、战斗、收集物品，体验每局不同的冒险旅程。
- 核心目标：提供高可玩性、高重玩价值的回合制地牢探索体验，让玩家沉浸在"再来一局"的循环中
- 目标用户：喜欢 Roguelike、像素风、策略类游戏的休闲及核心玩家

## 2. 核心功能

### 2.1 用户角色
不适用（单机游戏，无需注册登录）

### 2.2 功能模块
1. **主菜单页面**：游戏标题、开始游戏、操作说明、最高分记录
2. **游戏主页面**：地牢地图渲染、角色控制、战斗系统、物品系统、迷雾系统、HUD 信息面板
3. **游戏结束页面**：死亡/通关统计、分数展示、重新开始

## 3. 核心流程

玩家从主菜单开始游戏 → 进入随机生成的地牢第1层 → 探索房间、击杀敌人、收集物品 → 找到楼梯下到下一层 → 每5层遭遇Boss → 角色死亡则游戏结束，显示统计 → 可选择重新开始。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深灰黑（#1a1a2e）+ 暗紫（#16213e）+ 火焰橙（#e94560）作为强调色
- 次要色：暗金（#c9a227）用于物品/金币，暗绿（#2d6a4f）用于治疗/生命
- 按钮风格：像素风圆角按钮，带阴影和hover发光效果
- 字体：像素风字体（Press Start 2P）用于标题，等宽字体用于游戏内文字
- 布局：Canvas居中占主体，右侧/下方HUD信息面板
- 图标风格：像素图标，16x16或32x32网格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主菜单页面 | 标题区 | 大号像素字体标题，火焰粒子动效背景，暗色渐变底色 |
| 主菜单页面 | 按钮区 | 居中排列像素风按钮，hover发光，点击缩放反馈 |
| 主菜单页面 | 操作说明 | 半透明面板，图标+文字说明键位 |
| 游戏主页面 | 地牢画布 | Canvas居中，占主要区域，像素风地牢渲染 |
| 游戏主页面 | HUD顶栏 | 血量条（红）、经验条（蓝）、等级、楼层 |
| 游戏主页面 | 物品栏 | 底部横向排列，数字键快捷使用 |
| 游戏主页面 | 消息日志 | 右侧滚动文本区，战斗/拾取消息 |
| 游戏主页面 | 小地图 | 右上角缩略图，已探索区域灰显 |
| 游戏主页面 | 触控面板 | 移动端底部虚拟方向键+操作按钮 |
| 游戏结束页面 | 统计面板 | 居中面板，像素风边框，统计数据+分数 |
| 游戏结束页面 | 操作按钮 | 重新开始/返回主菜单 |

### 4.3 响应式设计
- 桌面优先设计，Canvas自适应窗口大小
- 移动端：Canvas缩放适配，显示虚拟方向键和操作按钮
- 触控优化：按钮足够大（≥48px），支持滑动移动

### 4.4 3D场景指引
不适用（2D像素风游戏）
