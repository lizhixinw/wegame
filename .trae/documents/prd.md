## 1. 产品概述

「暗影突击」是一款基于浏览器的横版卷轴射击游戏，灵感来自经典魂斗罗，玩家操控特种兵在充满敌人和陷阱的战场中冲锋陷阵，收集武器升级，击败Boss通关。
- 核心目标：提供紧张刺激、操作爽快的实时横版射击体验，让玩家沉浸在"再试一次"的动作游戏循环中
- 目标用户：喜欢动作射击、复古像素风、魂斗罗类游戏的休闲及核心玩家

## 2. 核心功能

### 2.1 用户角色
不适用（单机游戏，无需注册登录）

### 2.2 功能模块
1. **主菜单页面**：游戏标题动画、开始游戏、操作说明、最高分记录
2. **游戏主页面**：横版卷轴场景渲染、角色控制、射击系统、敌人系统、武器系统、HUD信息面板
3. **游戏结束页面**：通关/阵亡统计、分数展示、重新开始

## 3. 核心流程

玩家从主菜单开始游戏 → 进入第1关丛林基地 → 横向推进，射击敌人，跳跃平台，收集武器升级 → 击败关底Boss → 进入下一关 → 3条命用完则游戏结束，显示统计 → 可选择重新开始。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：军绿（#2d5016）+ 深棕（#3a2a1a）+ 火焰橙红（#ff4422）作为强调色
- 次要色：亮黄（#ffd700）用于武器道具，天蓝（#00bfff）用于激光武器
- 按钮风格：军事风硬朗按钮，金属质感边框，hover发光效果
- 字体：像素风字体（Press Start 2P）用于标题，等宽字体用于游戏内文字
- 布局：Canvas居中占主体，下方HUD信息面板
- 图标风格：像素军事图标，16x16或32x32网格
- 视觉特效：爆炸粒子、弹道轨迹、屏幕震动、闪光效果

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 主菜单页面 | 标题区 | 大号像素字体标题，爆炸粒子动效背景，军事渐变底色 |
| 主菜单页面 | 按钮区 | 居中排列军事风按钮，hover发光，点击缩放反馈 |
| 主菜单页面 | 操作说明 | 半透明面板，图标+文字说明键位 |
| 游戏主页面 | 游戏画布 | Canvas居中，占主要区域，像素风横版场景渲染 |
| 游戏主页面 | HUD顶栏 | 生命图标、武器类型图标、分数、关卡名称 |
| 游戏主页面 | 触控面板 | 移动端左侧虚拟摇杆+右侧射击/跳跃按钮 |
| 游戏结束页面 | 统计面板 | 居中面板，军事风边框，统计数据+分数 |
| 游戏结束页面 | 操作按钮 | 重新开始/返回主菜单 |

### 4.3 响应式设计
- 桌面优先设计，Canvas自适应窗口大小，保持16:9比例
- 移动端：Canvas缩放适配，显示虚拟摇杆和操作按钮
- 触控优化：按钮足够大（≥48px），支持多点触控

### 4.4 3D场景指引
不适用（2D像素风游戏）
